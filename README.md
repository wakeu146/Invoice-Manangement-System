# Invoice Management SaaS (PHP + MySQL)

## Overview

This is a **multi-tenant SaaS Invoice Management System** built with PHP and MySQL.  
It allows multiple companies to register, manage their own workspace, and handle invoices, clients, and payments securely.

The system is designed for **small businesses, freelancers, and academic projects**, and can be extended into a full SaaS product.

---

## Features

###  Multi-Company SaaS System
- Each company has its own isolated data
- Secure company-based dashboards
- Multiple users per company (Admin / Staff roles)

###  Invoice Management
- Create and manage invoices
- Auto invoice numbering
- Add products/services
- Automatic total calculation
- Discount support

###  Client Management
- Add, edit, delete clients
- View client invoice history
- Store contact details

### Payment Tracking
- Paid / Unpaid / Pending status
- Track outstanding balances
- Record payments

### PDF & Print Support
- Generate printable invoices
- Download invoices as PDF
- Clean professional invoice layout

### Authentication & Security
- Secure login system
- Password hashing
- Role-based access control
- Company-based data separation
- Custom error handling (403, 404, 500)

###  Dashboard
- Total invoices overview
- Revenue summary
- Recent activity tracking

---

## Tech Stack

- PHP (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- Apache Server (XAMPP / WAMP / Hosting compatible)
